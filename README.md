Si les configurations du compilateur ne sont pas actualisées pour le programme, tapez la commande suivante:

set path=%path%;-le chemin où se trouve le JDK-

Pour redimensionner l'image:
	- compiler le fichier java dans le fichier src avec la commande 			" javac SeamCarving.java "
	- puis taper:
	java SeamCarving [chemin de l'image] [%redimensionnement x]	[%redimensionnement y]
